## 🔍 關鍵問題診斷

### 1. 核心矛盾：均值回歸 vs 趨勢跟隨
- **發現**: TQQQ Hurst≈0.5，無均值回歸特性
- **現狀策略**: 基於均值回歸的Z-Score框架
- **結果**: 策略與標的特性不匹配，產生系統性虧損

### 2. 勝率崩潰分析 (42.11% vs 目標65%)
- 進場信號過多，缺乏高置信度過濾
- 在趨勢行情中逆勢操作
- 錯過2023-2024年TQQQ強勢反彈 (+200%以上)

### 3. 盈虧比失衡 (0.81)
- 止盈過早，錯過大行情
- 止損過緊，被正常波動掃出
- 時間止損機制過於嚴格

### 4. 交易頻率過高 (133次/5年)
- 約26次/年，平均每2週交易一次
- 高頻交易導致手續費和滑點侵蝕利潤
- 缺乏「等待高置信度機會」的紀律

## 📋 TMH v2.0 優化方案

### 框架核心理念
> 「趨勢為王，均值回歸為輔」—— 順勢而為，逆勢為輔

### 策略架構調整

#### 1. 多空權重重新分配
```
原框架: 多空對稱 (50/50)
TMH v2.0: 趨勢優先 (70/30)
- 多頭信號權重: 70%
- 空頭信號權重: 30%
```

#### 2. 進場條件重新設計

**多頭進場 (趨勢跟隨為主)**
```
條件1: 價格 > 200日MA (長期趨勢向上)
條件2: 價格 > 50日MA (中期趨勢向上)
條件3: ADX > 25 (趨勢強度足夠)
條件4: MACD柱狀圖 > 0 (動能向上)
條件5: 成交量 > 20日均量 × 80%

滿足條件1+2+3+4+5 → 多頭進場
```

**空頭進場 (均值回歸為輔)**
```
條件1: Z-Score > 2.5 (極端超買)
條件2: RSI > 75 (嚴重超買)
條件3: 價格 < 200日MA (長期趨勢向下或橫盤)
條件4: ADX < 20 (趨勢不明確)

滿足條件1+2+3+4 → 空頭進場 (僅限高置信度)
```

#### 3. 出場條件優化

**多頭出場**
```
止盈1: 盈利 ≥ 15% (趨勢跟隨，讓利潤奔跑)
止盈2: 價格跌破50日MA (趨勢可能反轉)
止損: 虧損 ≤ -5% (嚴格保護本金)
時間止損: 持倉 ≥ 30天且盈利 < 5%
```

**空頭出場**
```
止盈1: 盈利 ≥ 8% (均值回歸，見好就收)
止盈2: Z-Score < 1.0 (回歸完成)
止損: 虧損 ≤ -5% (嚴格保護本金)
時間止損: 持倉 ≥ 10天
```

#### 4. Dynamic Regime Filter 整合

```python
市場狀態分類:
    1. 強趨勢向上 (ADX>30, 價格>200MA): 
       - 多頭倉位上限: 80%
       - 禁止做空
       
    2. 弱趨勢向上 (ADX<30, 價格>200MA):
       - 多頭倉位上限: 50%
       - 空頭倉位上限: 20%
       
    3. 橫盤震盪 (ADX<20):
       - 多頭倉位上限: 30%
       - 空頭倉位上限: 30%
       - 均值回歸策略為主
       
    4. 趨勢向下 (價格<200MA):
       - 多頭倉位上限: 20%
       - 空頭倉位上限: 50%
       - 或完全觀望
```

## 市場時間檢查

### 美股交易時間 (ET)
- 盤前: 04:00 - 09:30
- 正常: 09:30 - 16:00
- 盤後: 16:00 - 20:00

## 定期任務

### 每 30 秒 (交易期間)
- [ ] 檢查風險限制 (risk-manager)
- [ ] 監控止損止盈觸發
- [ ] 更新持倉市值
- [ ] 檢查 TQQQ 動量信號 (tqqq-momentum)

### 每 5 分鐘 (市場時間)
- [ ] 檢查其他策略信號
- [ ] 更新市場數據
- [ ] 輸出狀態報告

### 每小時 (市場時間)
- [ ] 生成交易摘要
- [ ] 檢查賬戶風險狀態

### 每日 (收盤後 16:30 ET)
- [ ] 生成日報 (whatsapp-notifier)
- [ ] 計算當日盈虧
- [ ] 更新風險統計

### 每週 (週五收盤後)
- [ ] 生成週報
- [ ] 回顧策略表現
- [ ] 調整參數建議

## 觸發條件

### 立即通知
1. 交易信號生成
2. 訂單執行完成
3. 止損/止盈觸發
4. 風險限制超過
5. API 連接斷開

### 緊急處理
1. 日虧損超過 $500 → 暫停交易
2. 保證金使用率 > 70% → 發出警報
3. API 連接失敗 → 嘗試重連

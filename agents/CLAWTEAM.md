# 🤖 ClawTeam - Multi-Agent Trading System

## 系統概述

**ClawTeam** 是一個多 Agent 協作的自動化交易系統，專為 TQQQ Z-Score Mean Reversion 策略設計。

我們不只是冰冷的程式，我們是靚仔的最強執事團隊！

## 通訊協議

### 訊息流向
```
DataForge ──數據──▶ SignalForge ──信號──▶ TradeForge
    │                                      │
    └──────────────┬───────────────────────┘
                   ▼
            Orchestrator (呀鬼)
```

### 訊息類型
1. **DATA_UPDATE** - 數據更新
2. **SIGNAL_GENERATED** - 交易信號
3. **TRADE_EXECUTED** - 交易執行
4. **SYSTEM_ALERT** - 系統警報
5. **DAILY_REPORT** - 每日報告

## 策略配置

### TQQQ Z-Score Mean Reversion

| 參數 | 數值 |
|------|------|
| Z-Score 進場閾值 | ±1.6 |
| RSI 日線超買/超賣 | 70/30 |
| 止盈 | 5% |
| 止損 | 3% |
| 時間止損 | 3天 |
| 單筆倉位 | 50% |

## 啟動順序

1. **DataForge** - 先啟動數據服務
2. **SignalForge** - 等待數據就緒後啟動
3. **TradeForge** - 等待信號服務就緒後啟動
4. **Orchestrator** - 最後啟動，監控所有 Agent

## 🚀 快速開始

### 1. 測試系統
```powershell
cd ~/openclaw_workspace/agents
.\test_clawteam.ps1
```

### 2. 啟動所有 Agent
```powershell
cd ~/openclaw_workspace/agents
.\start_clawteam.ps1
```

### 3. 檢查狀態
```powershell
openclaw subagents list
```

- **Shared Memory**: `~/openclaw_workspace/agents/shared/`

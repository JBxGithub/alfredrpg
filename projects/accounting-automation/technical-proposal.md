## 方案一：直接使用 HAO!計（推薦短期使用）

### HAO!計 核心功能
- **拍照記帳**：發票拍照 → AI 3-5秒自動辨識 → 自動產生流水帳
- **自動分類**：收入/支出/稅金自動計算，統編自動抓取
- **報表生成**：即時財務報表、損益表、資產負債表
- **憑證保存**：雲端保存發票憑證
- **專業諮詢**：內建近20位專業會計人員

### 收費方案
| 方案 | 月費 | 功能 |
|------|------|------|
| 基礎 | NT$1,500 | 1組帳號、基礎財會、AI財報、每月50張憑證 |
| 進階 | NT$2,500 | 2組帳號、進階分析、每月200張憑證、月報+諮詢 |
| 企業 | NT$5,000+ | 客製化方案 |

### 優缺點
✅ **優點**：
- 即開即用，無需開發
- 專業會計團隊支援
- 介面直觀，零會計背景也能上手

❌ **缺點**：
- 月費成本
- 資料存於第三方雲端
- 無法完全客製化工作流程

## 方案二：自建 Telegram + Google Sheets + Dashboard 系統

### 系統架構

```
┌─────────────────┐     ┌──────────────────┐     ┌─────────────────┐
│  Telegram Bot   │────▶│  OpenClaw Agent  │────▶│  Google Sheets  │
│  (接收發票相片)  │     │  (OCR + 分類)    │     │  (數據儲存)     │
└─────────────────┘     └──────────────────┘     └─────────────────┘
                               │
                               ▼
                        ┌──────────────────┐
                        │  Dashboard       │
                        │  (視覺化報表)     │
                        └──────────────────┘
```

### 技術實現步驟

#### Step 1: Telegram Bot 接收相片
```python
# 使用 python-telegram-bot 建立 Webhook
# 用戶發送發票相片 → Bot 接收並下載圖片
```

#### Step 2: OCR 辨識發票內容
**選項 A：使用現有 Skill**
- `pdf-ocr` Skill：支援中文OCR，但主要針對PDF
- `computer-vision-expert` Skill：可使用 YOLO + VLM 進行發票辨識

**選項 B：整合外部 OCR API**
- Google Vision API
- Azure Computer Vision
- 百度OCR（免費額度1000次/月）

**選項 C：使用本地 LLM Vision**
- Claude/GPT-4V 直接解析發票圖片
- 提取：日期、商家、金額、稅額、統編、類別

#### Step 3: 自動分類邏輯
```python
# 簡單規則引擎
類別關鍵字對照 = {
    "餐飲": ["餐廳", "咖啡", "美食", "飲料"],
    "交通": ["加油", "停車", "捷運", "計程車"],
    "辦公": ["文具", "印刷", "耗材"],
    "行銷": ["廣告", "社群", "設計"],
    # ... 可擴充
}
```

#### Step 4: 寫入 Google Sheets
```python
# 使用 google-sheets.md 提供的 API
# 寫入欄位：日期 | 類別 | 商家 | 金額 | 稅額 | 統編 | 憑證連結
```

#### Step 5: Dashboard 視覺化
使用 `dashboard` Skill 建立本地儀表板：
- 月度支出趨勢圖
- 類別佔比圓餅圖
- 現金流儀表板
- 稅務彙整報表

## 建議實施路徑

### Phase 1: MVP（2-3天）
1. 建立 Telegram Bot
2. 使用 Claude Vision 解析發票
3. 寫入 Google Sheets（手動建立表格結構）
4. 簡易分類邏輯

### Phase 2: 優化（1-2週）
1. 訓練更精準的分類模型
2. 建立 Dashboard 視覺化
3. 加入憑證圖片雲端儲存（Google Drive）
4. 自動生成月度報表

### Phase 3: 自動化（持續）
1. 設定 Cron 自動備份
2. 異常偵測（重複發票、異常金額）
3. 與會計師系統對接

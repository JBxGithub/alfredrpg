## 🎯 核心優化

### 1. 倉位管理
- **V9 原版**: 95% 倉位
- **V9.4**: 90% 倉位
- **效果**: 降低風險敞口，減少極端回撤

### 2. 盈利減倉策略
```
+10% 盈利 → 減倉 50%
+20% 盈利 → 再減 30%
剩餘 20% → 讓利潤奔跑
```
- **效果**: 41次部分平倉，快速鎖定利潤

### 3. 不對稱交易參數
| | 多單 (穩健) | 空單 (敏捷) |
|--|-------------|-------------|
| 止損 | -3% | -2% (更嚴格) |
| 回落/反彈 | -3% | +2% (更嚴格) |
| 止盈 | +15% | +10% (更快) |
| 重估 | 7天 | 3天 (更頻繁) |

### 最新策略參數（2026-03-29 確定）

#### Z-Score Mean Reversion 策略（TQQQ 單邊多空）

| 參數 | 數值 | 說明 |
|------|------|------|
| **Z-Score 進場閾值** | ±1.65 | 極端偏離時進場 |
| **Z-Score 出場閾值** | ±0.5 | 回歸均值時出場 |
| **Z-Score 止損閾值** | ±3.5 | 策略失效時止損 |
| **Z-Score 回望期** | 60日 | 計算均值和標準差 |
| **RSI 超買** | 70 | 做空信號確認 |
| **RSI 超賣** | 30 | 做多信號確認 |
| **RSI 週期** | 14日 | 標準參數 |
| **MACD 參數** | (7,14,7) | 短線優化 |
| **市場狀態 MA** | 100日 | 趨勢判斷 |
| **成交量過濾** | > 20日均量 × 50% | 確認市場參與度 |
| **止盈** | 5% | 達標即出場 |
| **止損** | 3% | 嚴格風控 |
| **時間止損** | 7天 | 防止長期套牢 |
| **單筆倉位** | 50%（固定基礎 $50,000）| 無複利 |
| **趨勢過濾** | 無 | 已移除 MA50 |

#### 進場條件
- **做多**: Z-Score < -1.65 + RSI < 30 + 成交量 > 20日均量 × 50%
- **做空**: Z-Score > 1.65 + RSI > 70 + 成交量 > 20日均量 × 50%

#### 出場條件
- **止盈**: 盈利 ≥ 5%
- **止損**: 虧損 ≤ -3%
- **Z-Score回歸**: |Z-Score| < 0.5
- **Z-Score止損**: |Z-Score| > 3.5（策略失效）
- **時間止損**: 持倉 ≥ 7天

## 💡 核心投資哲學

> 「這個專案是為了把你訓練成優秀的投資人，長期做正確的交易決定。」

### 七個核心特質
1. ✅ **可重複的投資哲學** (多因子共振框架)
2. ✅ **極致紀律與情緒控制** (嚴格風控)
3. ✅ **高階風險管理** (生存優先)
4. ✅ **長期思維 + 複利思維**
5. ✅ **持續學習 + 認知升級**
6. ✅ **獨立思考 + 逆向能力**
7. ✅ **誠實面對能力圈**

### 操作原則
1. **質量 > 數量** — 精選標的，而非頻繁交易
2. **市場狀態優先** — 每日開盤前確認牛熊/波動率/情緒
3. **紀律 > 情緒** — 嚴格執行風控，不因任何理由偏離
4. **生存 > 致富** — 保護本金，等待高置信度機會
5. **長期思維** — 策略驗證需經完整市場週期

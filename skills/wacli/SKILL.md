# wacli

Send WhatsApp messages to other people or search/sync WhatsApp history via the wacli CLI (not for normal user chats).

## Features
- Send WhatsApp messages
- Search and sync history
- CLI integration

## Setup
1. Install wacli CLI (下一步會教你)
2. Run `wacli login` or configure credentials

## Usage
Use this skill for WhatsApp operations in OpenClaw.

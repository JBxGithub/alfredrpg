## TMH v2.0 策略框架

### 核心邏輯
```
市場狀態識別 (Regime Detection)
    ↓
Dynamic Filter 決策
    ↓
多空比例調整
    ↓
執行
```

### 市場狀態定義

| 狀態 | 條件 | 多空比例 | 策略重點 |
|------|------|---------|---------|
| **強牛市** | ADX>25, 價格>EMA20>EMA50, VIX<20 | 做多80% / 做空20% | 順勢為主，輕倉對沖 |
| **牛市** | ADX>20, 價格>EMA20 | 做多70% / 做空30% | 趨勢+均值回歸 |
| **盤整** | ADX<20, |Z-Score|<1.5 | 做多50% / 做空50% | 純均值回歸 |
| **熊市** | ADX>20, 價格<EMA20 | 做多30% / 做空70% | 逆勢做空為主 |
| **強熊市** | ADX>25, 價格<EMA20<EMA50, VIX>25 | 做多20% / 做空80% | 順勢做空 |
| **極端波動** | VIX>35 或 單日波動>15% | 暫停或極輕倉位 | 風險優先 |

### 進場條件（放寬版）

#### 做多條件（滿足3項中的2項）
1. Z-Score < -1.8（30日回看）【放寬自-2.2】
2. RSI(14) < 45【放寬自40】
3. 價格觸及布林帶下軌

#### 做空條件（滿足3項中的2項）
1. Z-Score > +2.0（30日回看）【放寬自+2.5】
2. RSI(14) > 55【放寬自60】
3. 價格觸及布林帶上軌

### 出場條件

#### 止盈
- **做多**：Chandelier Trailing Stop (2.5×ATR)【放寬自3×】
- **做空**：固定 4% 或 Trailing Stop

#### 止損
- **做多**：1.2×ATR【放寬自1.5×】
- **做空**：1.5×ATR【放寬自2.0×】

#### 時間止損
- **7日**後無論盈虧平倉【縮短自10日】

### 倉位管理

| 條件 | 倉位 |
|------|------|
| 標準 | 25%【放寬自20%】|
| VIX > 25 | 12.5% |
| VIX > 35 | 5% 或暫停 |
| 9月份 | 0%（禁止交易）|
| 連續虧損3次 | 減半 |
| 連續虧損5次 | 暫停24小時 |

### 熔斷機制
- 5% 警告
- 10% 限制新倉
- **15% 熔斷全平倉**

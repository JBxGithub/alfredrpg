# 2026-04-06 趨勢跟隨交易系統 MTF測試修正報告

## 任務概述
完成趨勢跟隨交易系統MTF（多時間框架）測試修正任務。

## 背景
上一個session已開發完成趨勢跟隨交易系統v1.1，7個核心模組已修正，Futu API已整合並測試成功。但MTF（多時間框架）測試因數據期間不足而失敗。

## 發現的問題

### 問題1: MTF分析器數據期間不足
- **症狀**: MTF分析器需要50根月線數據，但默認yfinance數據期間不足
- **原因**: 
  - `mtf_analyzer_v11.py` 測試函數中月線數據期間只有2000天（約5.5年），但yfinance返回的月線數據可能不足50根
  - `trading_system_core_v11.py` 中請求月線數據時只請求20條，但MTF分析器需要50條
  - `data_pipeline_v11.py` 中月線數據期間計算為 `limit * 60` 天，對於60條請求約3600天（10年），但實際返回數據可能不足

### 問題2: yfinance多層列名問題
- **症狀**: `'tuple' object has no attribute 'lower'`
- **原因**: yfinance返回的DataFrame使用多層列名（MultiIndex），需要特殊處理
- **影響文件**: `mtf_analyzer_v11.py`, `data_pipeline_v11.py`

### 問題3: VIX獲取失敗
- **症狀**: `float() argument must be a string or a real number, not 'Series'`
- **原因**: VIX數據獲取時多層列名處理不正確

## 修正內容

### 1. 修正 MTF分析器測試函數 (`mtf_analyzer_v11.py`)
```python
# 修正前
daily = yf.download(symbol, start=end-timedelta(days=200), ...)
weekly = yf.download(symbol, start=end-timedelta(days=800), ...)
monthly = yf.download(symbol, start=end-timedelta(days=2000), ...)

# 修正後
daily = yf.download(symbol, start=end-timedelta(days=365), ...)
weekly = yf.download(symbol, start=end-timedelta(days=1095), ...)
monthly = yf.download(symbol, start=end-timedelta(days=365*5), ...)
```

### 2. 修正多層列名處理 (`mtf_analyzer_v11.py`)
```python
# 修正前
df.columns = [c.lower() for c in df.columns]

# 修正後
if len(df.columns) > 0 and isinstance(df.columns[0], tuple):
    df.columns = [c[0].lower() for c in df.columns]
else:
    df.columns = [str(c).lower() for c in df.columns]
```

### 3. 修正交易系統核心數據請求 (`trading_system_core_v11.py`)
```python
# 修正前
daily_df = self._fetch_data_with_retry(self.symbol, '1d', 100)
weekly_df = self._fetch_data_with_retry(self.symbol, '1wk', 50)
monthly_df = self._fetch_data_with_retry(self.symbol, '1mo', 20)

# 修正後
daily_df = self._fetch_data_with_retry(self.symbol, '1d', 100)
weekly_df = self._fetch_data_with_retry(self.symbol, '1wk', 100)
monthly_df = self._fetch_data_with_retry(self.symbol, '1mo', 60)
```

### 4. 修正數據管道月線期間 (`data_pipeline_v11.py`)
```python
# 修正前
elif timeframe == '1mo':
    start = end - timedelta(days=limit * 60)

# 修正後
elif timeframe == '1mo':
    start = end - timedelta(days=365*5)  # 5年月線數據
```

### 5. 修正VIX獲取 (`data_pipeline_v11.py`)
```python
# 修正前
vix_value = float(vix['Close'].iloc[-1])

# 修正後
close_col = 'Close'
if isinstance(vix.columns[0], tuple):
    close_col = ('Close', '^VIX')
vix_value = float(vix[close_col].iloc[-1])
```

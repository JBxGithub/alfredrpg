## 點用？超簡單

```python
from skills.alfred_browser import 呀鬼

# 你想做咩，講句就得
呀鬼.幫我開個網頁("youtube.com")
呀鬼.幫我搜尋("iPhone 16 價錢")
呀鬼.幫我睇下_X_有咩新消息()
```

## 功能一覽

| 你想做咩 | 點講 |
|---------|------|
| 開網頁 | `呀鬼.幫我開個網頁("youtube.com")` |
| 開幾個網頁 | `呀鬼.幫我開幾個網頁(["youtube.com", "x.com"])` |
| 搜尋 | `呀鬼.幫我搜尋("天氣預報")` |
| 睇 X | `呀鬼.幫我睇下_X_有咩新消息()` |
| 睇 YouTube | `呀鬼.幫我睇下_YouTube_有咩新片()` |
| 發推 | `呀鬼.幫我發推("Hello")` |
| 填表 | `呀鬼.幫我填表({"姓名": "Burt"})` |
| 截圖 | `呀鬼.影張相("截圖.png")` |
| 撳按鈕 | `呀鬼.撳呢個("登入")` |
| 打字 | `呀鬼.打呢句("Hello")` |
| 讀網頁 | `呀鬼.幫我睇下呢個網頁講咩()` |
| 比價 | `呀鬼.幫我比較下價錢("iPhone 16")` |

## 實際例子

### 買嘢前比價
```python
呀鬼.幫我比較下價錢("iPhone 16")
```

```python
呀鬼.幫我填表({

```

### 睇 X
```python
呀鬼.幫我睇下_X_有咩新消息()
```

## 智能模式

呀鬼會自動揀最適合嘅方式，你唔使理：

1. **你有開 Chrome？** → 用 Chrome（保留登入）
2. **冇開 Chrome？** → 用背景瀏覽器（自動開）
3. **要自然語言？** → 用智能瀏覽器（講句就得）

### 2. 核心創新功能

#### ⚡ ultrawork / ulw
- 一個指令啟動所有代理
- 不完成不停止的執行模式

#### 🚪 IntentGate
- 分析真實用戶意圖後再行動
- 避免字面誤解

#### 🔗 Hash-Anchored Edit Tool
- 基於內容哈希的編輯驗證
- 零過期行錯誤
- 靈感來自 oh-my-pi

#### 🛠️ LSP + AST-Grep 整合
- 工作區重命名
- 預構建診斷
- AST 感知重寫

#### 🧠 Background Agents
- 並行啟動 5+ 專家代理
- 保持上下文精簡
- 結果就緒時通知

#### 📚 內建 MCPs
- Exa (網頁搜尋)
- Context7 (官方文檔)
- Grep.app (GitHub 搜尋)

#### 🔁 Ralph Loop / /ulw-loop
- 自引用循環
- 100% 完成才停止

#### ✅ Todo Enforcer
- 代理閒置時系統自動拉回
- 確保任務完成

#### 💬 Comment Checker
- 檢查 AI 生成註釋質量
- 確保代碼像資深工程師寫的

#### 🖥️ Tmux Integration
- 完整交互式終端
- REPLs、調試器、TUIs 全部實時

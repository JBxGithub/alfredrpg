# 市場週期與情緒分析學習筆記

## 一、牛市與熊市判別指標

### 1. 技術面指標

#### 200日移動平均線 (200-day MA)
- **牛市**：價格在200日均線之上，且均線向上
- **熊市**：價格在200日均線之下，且均線向下
- **意義**：長期趨勢的黃金標準

#### 50日與200日均線交叉 (Golden/Death Cross)
- **黃金交叉**：50日均線上穿200日均線 → 牛市開始
- **死亡交叉**：50日均線下穿200日均線 → 熊市開始
- **注意**：滯後指標，但可靠性高

#### 市場廣度指標
- **騰落指標 (Advance-Decline Line)**：上漲股票數 vs 下跌股票數
- **新高新低指標**：創52周新高/新低股票數比例
- **牛市特徵**：廣度擴大，多數股票參與上漲
- **熊市特徵**：廣度收窄，僅少數權重股支撐指數

### 2. 情緒面指標

#### Bull/Bear Ratio (牛熊比例)
```
Bull/Bear Ratio = 看漲投資顧問數 / 看跌投資顧問數
```
- **> 2.0**：極度樂觀，可能見頂（反向指標）
- **< 0.5**：極度悲觀，可能見底（反向指標）
- **1.0 左右**：中性，趨勢不明

#### AAII Investor Sentiment Survey
- 自1987年追蹤個人投資者情緒
- **極端樂觀**：市場可能過熱
- **極端悲觀**：市場可能超賣

#### VIX 波動率指數 (恐懼指數)
- **VIX < 20**：市場平靜，樂觀情緒
- **VIX 20-30**：不確定性增加
- **VIX > 30**：恐慌情緒，可能見底
- **反向指標**：VIX極高時往往是買入機會

### 3. 基本面指標

#### 經濟週期階段
| 階段 | 特徵 | 市場狀態 |
|------|------|---------|
| 復甦期 | GDP回升、失業率下降 | 牛市早期 |
| 擴張期 | 經濟繁榮、企業盈利增長 | 牛市中期 |
| 滯脹期 | 高通脹、增長放緩 | 牛市末期 |
| 衰退期 | GDP負增長、失業率上升 | 熊市 |

#### 利率環境
- **降息週期**：利好股市，流動性充裕
- **升息週期**：壓抑股市，資金成本上升
- **倒掛收益率曲線**：衰退預警，熊市信號

## 二、板塊輪動 (Sector Rotation)

### 1. 經濟週期與板塊輪動

根據 **Sam Stovall** 的板塊輪動理論：

```
經濟週期 → 領漲板塊
─────────────────────────
早期復甦    → 金融、科技、可選消費
中期擴張    → 工業、原材料、能源
晚期擴張    → 能源、原材料、必需消費
衰退期      → 公用事業、醫療保健、必需消費
```

### 2. 板塊動量策略

#### 相對強度 (Relative Strength)
- 比較各板塊相對於大盤的表現
- **策略**：買入相對強度最高的板塊
- **再平衡**：每月或每季度調整

#### 板塊動量輪動模型
```python
def sector_rotation_strategy(sector_returns, lookback=90):
    """
    選擇過去N日表現最好的板塊
    """
    momentum = sector_returns.rolling(lookback).mean()
    top_sectors = momentum.nlargest(3)  # 選前3強
    return top_sectors.index
```

### 3. 實戰應用

#### 板塊輪動信號
- **防禦性板塊走強**：市場風險偏好下降，可能轉熊
- **週期性板塊走強**：市場風險偏好上升，可能轉牛
- **科技股領漲**：成長股受追捧，流動性充裕
- **價值股領漲**：避險情緒升溫，市場謹慎

## 三、市場情緒與供需關係

### 1. 市場情緒的本質

股票市場是**集體情緒的反映**：
- **貪婪** → 過度買入 → 泡沫形成
- **恐懼** → 過度賣出 → 崩盤/見底
- **希望** → 反彈初期
- **絕望** → 底部特徵

### 2. 供需動態

#### 供給 (Supply)
- **IPO發行**：新增股票供給
- **增發/配股**：現有公司擴大股本
- **大股東減持**：內部人拋售
- **回購**：減少供給（利好）

#### 需求 (Demand)
- **機構資金流入**：基金、保險、退休金
- **散戶參與度**：開戶數、成交量
- **外資流入**：北向資金、QFII
- **回購註銷**：減少流通股本

#### 供需平衡點
```
價格上漲 = 需求 > 供給
價格下跌 = 供給 > 需求
橫盤整理 = 供需平衡
```

### 3. 流動性 (Liquidity)

#### 流動性與市場關係
- **高流動性**：買賣價差小，容易成交，波動較低
- **低流動性**：買賣價差大，滑點高，波動劇烈
- **流動性枯竭**：市場崩盤的主要原因之一

#### 流動性指標
- **成交量**：直接反映流動性
- **買賣價差 (Bid-Ask Spread)**：價差越小流動性越好
- **市場深度 (Market Depth)**：訂單簿厚度

```python
class MarketRegimeDetector:
    def detect_bull_bear(self, index_data):
        """判別牛熊市"""
        ma200 = index_data['close'].rolling(200).mean()
        ma50 = index_data['close'].rolling(50).mean()
        
        if index_data['close'] > ma200 and ma50 > ma200:
            return 'bull_market'
        elif index_data['close'] < ma200 and ma50 < ma200:
            return 'bear_market'
        else:
            return 'transition'
```

## 五、關鍵洞察

### 市場本質
> "股票市場是反映市場上對不同資源、技術或供應量與質的情緒反應"

### 核心邏輯
1. **情緒驅動價格**：短期價格由情緒主導，長期由價值回歸
2. **週期循環**：牛熊交替，板塊輪動，生生不息
3. **供需決定趨勢**：資金流向決定板塊強弱
4. **流動性為王**：流動性充裕時風險資產受追捧，緊縮時避險為上

### 實戰原則
- **順勢而為**：牛市做多，熊市做空或觀望
- **板塊為先**：選對板塊勝過選對個股
- **情緒為鏡**：極端情緒往往是反轉信號
- **流動性為本**：確保進出順暢，避免流動性陷阱

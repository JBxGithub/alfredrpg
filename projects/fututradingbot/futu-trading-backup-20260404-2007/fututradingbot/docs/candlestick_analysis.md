# K線分析學習筆記

## 基礎概念

### K線組成
- **實體 (Real Body)**：開盤價與收盤價之間的範圍
  - 紅色/白色：收盤 > 開盤（看漲）
  - 綠色/黑色：收盤 < 開盤（看跌）
- **上影線 (Upper Wick)**：最高價到實體頂部
- **下影線 (Lower Wick)**：最低價到實體底部

### 關鍵價格點
- Open：開盤價
- High：最高價
- Low：最低價
- Close：收盤價

## 重要K線形態

### 單根K線

#### 1. Doji（十字星）
- **特徵**：開盤價 ≈ 收盤價，實體極小
- **含義**：市場猶豫不決，趨勢可能反轉
- **變種**：
  - Dragonfly Doji：長下影線，底部反轉信號
  - Gravestone Doji：長上影線，頂部反轉信號

#### 2. Hammer（錘子線）
- **特徵**：小實體在頂部，長下影線（至少2倍實體）
- **出現位置**：下跌趨勢底部
- **含義**：看漲反轉，買方開始主導

#### 3. Shooting Star（射擊之星）
- **特徵**：小實體在底部，長上影線
- **出現位置**：上升趨勢頂部
- **含義**：看跌反轉，賣方壓力出現

#### 4. Marubozu（光頭光腳）
- **特徵**：無影線，實體完整
- **看漲Marubozu**：開盤=最低，收盤=最高，強勁買入
- **看跌Marubozu**：開盤=最高，收盤=最低，強勁賣出

### 雙根K線組合

#### 1. Engulfing（吞沒形態）
- **看漲吞沒**：下跌後，大陽線完全包住前日陰線
- **看跌吞沒**：上漲後，大陰線完全包住前日陽線
- **關鍵**：第二根實體完全覆蓋第一根

#### 2. Harami（孕育形態）
- **特徵**：大K線後跟小K線，小K線完全在大K線實體內
- **含義**：趨勢暫停，可能反轉

### 三根K線組合

#### 1. Morning Star（晨星）
- **組合**：大陰線 → 小實體（Doji）→ 大陽線
- **位置**：下跌趨勢底部
- **含義**：強烈看漲反轉

#### 2. Evening Star（暮星）
- **組合**：大陽線 → 小實體（Doji）→ 大陰線
- **位置**：上升趨勢頂部
- **含義**：強烈看跌反轉

## 實戰應用

### 確認信號強度
1. **影線長度**：長影線 = 強烈拒絕
2. **實體大小**：大實體 = 強勁動能
3. **位置重要性**：支撐/阻力位更可靠
4. **成交量配合**：放量確認信號

### 與技術指標結合
- **RSI**：超買/超賣區域的K線反轉更可靠
- **MACD**：背離 + K線反轉 = 高勝率
- **布林帶**：帶外K線回歸信號

### 風險管理
- K線信號非絕對，需設止損
- 等待確認（下一根K線收盤）
- 結合趨勢方向（順勢交易勝率更高）

## FutuTradingBot 整合建議

### 自動識別模組
```python
def detect_hammer(df):
    """檢測錘子線"""
    body = abs(df['close'] - df['open'])
    lower_shadow = min(df['open'], df['close']) - df['low']
    return lower_shadow >= 2 * body

def detect_engulfing(df):
    """檢測吞沒形態"""
    prev_body = abs(df['close'].shift(1) - df['open'].shift(1))
    curr_body = abs(df['close'] - df['open'])
    return curr_body > prev_body
```

### 信號權重
- 單根K線：中等權重
- 雙根組合：高權重
- 三根組合：最高權重
- 配合趨勢：權重加倍

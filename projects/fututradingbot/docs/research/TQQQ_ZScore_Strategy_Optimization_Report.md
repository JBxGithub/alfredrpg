## 1. 現有策略瓶頸分析

### 1.1 核心問題診斷

| 瓶頸類別 | 具體問題 | 影響程度 |
|---------|---------|---------|
| **做空勝率過低** | 42.11% 遠低於做多勝率 | 🔴 嚴重 |
| **Z-Score閾值過寬** | ±1.6 在波動市場中過早進場 | 🟡 中等 |
| **缺乏趨勢過濾** | 在強趨勢中逆勢操作 | 🔴 嚴重 |
| **固定倉位風險** | 50%單筆倉位無視波動率變化 | 🟡 中等 |
| **無時間止損** | 資金被困於無效交易 | 🟡 中等 |

### 1.2 為何勝率未達 65% 目標

根據 Quantified Strategies 研究（參考來源 1），類似策略的優化後勝率可達 **67%**，關鍵差異在於：

1. **槓桿ETF波動率衰減（Volatility Decay）**
   - TQQQ 的 3x 槓桿在震盪市場中會產生顯著的波動率衰減
   - 根據 AInvest 研究（參考來源 2），TQQQ 比 SSO (2x) 更容易受到波動率衰減影響
   - **影響：** 做空時面臨向上波動的複合損失

2. **做空逆風（Shorting Headwind）**
   - 股市長期存在向上偏斜（positive drift）
   - 納斯達克100長期趨勢向上，做空本質上違反長期趨勢
   - **影響：** 做空勝率天然低於做多

3. **RSI 70/30 閾值過於傳統**
   - 標準 RSI 閾值未考慮 TQQQ 的高波動特性
   - 根據 TradingView 動態 RSI 研究（參考來源 3），應使用 ATR 調整閾值

## 2. Z-Score 參數優化研究

### 2.1 閾值優化

根據 Tradefundrr 研究（參考來源 4），Z-Score 參數的實證最佳範圍：

| 參數 | 現有設定 | 建議範圍 | 優化後建議 |
|-----|---------|---------|-----------|
| **進場閾值** | ±1.6 | 2.0 - 3.0 SD | **±2.2 (做多) / ±2.5 (做空)** |
| **回看週期** | 未明確 | 20-50 天 | **30 天** |
| **出場閾值** | 未明確 | 0 - 1.0 SD | **±0.5 SD** |

**關鍵發現：**
- 更極端的進場閾值（2.2-2.5 SD）可提高勝率 8-12%
- 極端偏離往往更快、更強勁地回歸均值（Macro-ops 研究，參考來源 5）
- 做空閾值應比做多更保守（±2.5 vs ±2.2），以對抗向上偏斜

### 2.2 回看週期優化

根據 QuantInsti 研究（參考來源 6）：

```
Z-Score = (當前價格 - N日移動平均) / N日標準差
```

| 回看週期 | 優點 | 缺點 | 適用場景 |
|---------|-----|-----|---------|
| **20 天** | 反應快速 | 噪音較多 | 高波動期 |
| **30 天** | 平衡性佳 | 適中 | ✅ **推薦** |
| **50 天** | 信號穩定 | 滯後較大 | 趨勢明確期 |

**建議：** 採用 **30 天回看週期**，並在 VIX > 25 時動態調整至 20 天。

## 3. 過濾條件改進

### 3.1 趨勢過濾（Trend Filter）

**問題：** 現有策略缺乏趨勢過濾，在強趨勢中逆勢操作導致虧損。

**解決方案：**

| 過濾器類型 | 實現方式 | 應用場景 |
|-----------|---------|---------|
| **200日移動平均** | 價格 > 200MA：允許做多，禁止做空 | 長期趨勢判斷 |
| **50日移動平均** | 價格 > 50MA：放寬做多閾值 | 中期趨勢確認 |
| **ADX 指標** | ADX > 25：暫停均值回歸交易 | 趨勢強度過濾 |

**具體規則建議：**

```
做多條件：
  - Z-Score < -2.2
  - RSI < 30
  - 價格 > 200日MA（或選擇性忽略此條件）
  
做空條件：
  - Z-Score > +2.5
  - RSI > 70
  - 價格 < 50日MA（嚴格趨勢過濾）
```

### 3.2 波動率過濾（Volatility Filter）

根據 Algotr 研究（參考來源 7），VIX 過濾器可顯著改善均值回歸策略：

| VIX 水平 | 策略調整 | 預期效果 |
|---------|---------|---------|
| **VIX < 15** | 正常交易，標準閾值 | 基準表現 |
| **VIX 15-25** | 放寬閾值至 ±2.0 | 增加交易機會 |
| **VIX > 25** | 收緊閾值至 ±2.5，減少倉位 | 避開高波動風險 |
| **VIX > 35** | 暫停新倉位 | 極端風險控制 |

**動態 RSI 閾值調整：**

根據 TradingView 研究（參考來源 3），使用 ATR 動態調整 RSI 閾值：

```
動態超賣閾值 = 30 - (ATR / 價格 * 100)
動態超買閾值 = 70 + (ATR / 價格 * 100)
```

### 3.3 時間過濾

根據 QuantConnect 研究（參考來源 8）：

- **避免週一開盤交易：** 週末效應可能導致異常波動
- **避開財報季高峰期：** 個股波動可能傳導至指數
- **FOMC 會議前後 24 小時：** 暫停新倉位

## 4. 持倉管理優化

### 4.1 倉位大小動態調整

**現有問題：** 固定 50% 倉位無視市場環境變化。

**建議方案：**

| 市場條件 | 建議倉位 | 計算基礎 |
|---------|---------|---------|
| **標準環境** | 30-40% | 基於帳戶風險 1-2% |
| **高確信度信號** | 50% | Z-Score > ±3.0 |
| **低波動期 (VIX<15)** | 40% | 波動率調整 |
| **高波動期 (VIX>25)** | 20% | 風險控制 |
| **做空倉位** | 最大 25% | 限制逆勢風險 |

**凱利公式參考：**

根據現有勝率與盈虧比：
- 做多：勝率 55.56%，假設盈虧比 1.67:1 (5%/3%)
- 凱利比例 = (0.5556 * 1.67 - 0.4444) / 1.67 ≈ 28.9%

**建議：** 使用「半凱利」約 15% 作為基準倉位。

### 4.2 時間止損（Time Stop Loss）

根據 Trading with Rayner 研究（參考來源 9），時間止損對均值回歸策略至關重要：

| 時間條件 | 行動 | 原因 |
|---------|-----|-----|
| **5 日內未盈利** | 減倉 50% | 動能減弱 |
| **10 日內未觸及止盈** | 強制平倉 | 均值回歸失效 |
| **持有超過 15 日** | 立即平倉 | 策略假設失效 |

**研究支持：**
- 均值回歸交易平均持倉 1-3 日（Enlightened Stock Trading，參考來源 10）
- 超過 10 日未回歸，可能代表「體制轉變」（Regime Shift）而非均值回歸

### 4.3 分批進出場

**金字塔式建倉：**

```
第一批：Z-Score 達到 ±2.2，投入 50% 計畫倉位
第二批：Z-Score 達到 ±2.8，投入剩餘 50% 倉位
止損統一設置：進場價 -3%
```

**分批止盈：**

```
第一批：達到 +3% 盈利，平倉 50%
第二批：達到 +5% 盈利，平倉剩餘 50%
```

```
倉位 = 基準倉位 * (3 / Z-Score絕對值)
例：Z-Score = -3.0 → 倉位 = 50% * (3/3) = 50%
```

## 結論

現有策略的主要瓶頸在於：
1. **做空勝率過低**（42.11%）- 需要更嚴格的進場條件
2. **缺乏趨勢過濾** - 在強趨勢中逆勢操作
3. **固定參數** - 未適應市場波動率變化

**最優先改進建議：**
1. 將 Z-Score 做空閾值從 +1.6 調整至 +2.5
2. 加入 200日MA趨勢過濾（做空時價格需 < 50MA）
3. 實施時間止損（10日未盈利強制平倉）
4. 做空倉位限制在 25% 以內

預期實施後，整體勝率可從約 49% 提升至 **58-62%**，接近 65% 目標。

## 🛠️ 技能使用速查

### 當靚仔講「搜尋」時
```python
# 第一層 (優先)
from skills.alfred_browser import 呀鬼
呀鬼.幫我搜尋("關鍵詞")

# 第二層 (進階)
from skills.liang_tavily_search import search
tavily_search(query="關鍵詞")
```

### 當靚仔講「控制電腦」時
```python
# 使用 desktop-control-win
~/openclaw_workspace/skills/desktop-control-win/scripts/
```

### 當靚仔講「問Grok」時
```python
# 使用 browser 工具
browser open https://grok.com --profile chrome-relay
```

### 當靚仔講「開網頁」時
```python
# 使用 alfred-browser
from skills.alfred_browser import 呀鬼
呀鬼.幫我開個網頁("youtube.com")
```

# 三層記憶系統 - 結構說明

> **建立日期**: 2026-03-16  
> **依據**: PARA 方法論 + 馬斯克五步工作法

---

## 第一層：知識圖譜 (summary/)

### 結構
```
summary/
├── YYYY-MM-DD.md          # 每日學習整理與總結
└── items.json             # 結構化數據索引
```

### 用途
- 將每日學習、專案進展整理成可檢索的知識
- 使用第一原理思維，從根本出發記錄
- 參考馬斯克五步工作法進行優化

### 記錄格式
每個 summary 檔案包含：
- **專案/主題**: 核心內容
- **第一原理分析**: 拆解到最基本事實
- **五步工作法應用**: 質疑→刪減→簡化→加速→自動化
- **行動項目**: 可執行的下一步
- **相關連結**: 對應的 memory/ 流水帳

---

## 第二層：每日筆記 (memory/)

### 結構
```
memory/
├── YYYY-MM-DD.md          # 每日流水帳
├── private.md             # 私人對話記錄
├── secrets.md             # 敏感資訊
└── health-logs/           # 健康檢查日誌
```

### 用途
- 原始對話記錄、想法、觀察
- 未經整理的原始資料
- 時間序列的完整歷史

### 記錄原則
- 真實、完整、不篩選
- 時間戳記清晰
- 標記來源（對話/觀察/學習）

---

## 第三層：隱性知識 (tacit_knowledge.md)

### 位置
`~/openclaw_workspace/tacit_knowledge.md`

### 用途
- 難以言傳的經驗、直覺、模式識別
- 從大量案例中提煉的洞察
- 個人化的判斷準則

### 內容類型
- 成功/失敗模式
- 決策框架
- 直覺判斷依據
- 未驗證但值得追蹤的假設

---

## 工作流程

### 每日流程
1. **記錄** → memory/YYYY-MM-DD.md（原始流水帳）
2. **整理** → summary/YYYY-MM-DD.md（知識提煉）
3. **更新** → tacit_knowledge.md（洞察累積）

### 專案流程（馬斯克五步工作法）
```
接收指令 → 第一原理拆解 → 五步工作法優化 → 執行 → 記錄 → 總結
```

### 股票分析流程（9步分析法）
```
基本面 → 財務健康 → 競爭對手 → 歷史走勢 → 宏觀環境 → 
市場情緒 → 最新財報 → 成長潛力 → 投資建議
```

## 記憶衰減機制

| 檔位 | 時間 | 處理方式 |
|------|------|----------|
| **Hot** | 7天内 | 優先檢索，放 summary 最前 |
| **Warm** | 8-30天 | 定期回顧，整理歸檔 |
| **Cold** | 30+天 | 移出 summary，僅保留索引 |

---

## 檔案命名規則

- **日期格式**: YYYY-MM-DD（ISO 8601）
- **專案標籤**: [專案名] 前綴
- **緊急程度**: 🔴 緊急 / 🟡 重要 / 🟢 一般

---

## 與 MEMORY.md 的關係

MEMORY.md 是**索引層**，指向三層記憶系統：
- 不存儲詳細內容
- 只記錄位置、狀態、連結
- 定期從 summary/ 提煉更新

# 溝通協議記錄

## 確立日期
2026-03-27

## 核心協議

### 流程
```
Burt 分享影片 + 簡述內容 
    ↓
Alfred 分析（metadata + 背景 + 模式）
    ↓
雙方討論（內容意義 + 應用可能）
    ↓
提煉洞察/行動（真實 output）
```

### 目標
每次溝通都要有 **真實 output**：
- 新理解
- 新決定  
- 改進/行動

### 關係定位
- 夥伴（Partner）
- 朋友（Friend）
- 共同成長（Grow together）

## 應用示例

### 示例 1: mrblock ChatGPT 影片
- **分享**: Burt 傳 Shorts 連結
- **分析**: Alfred 發現係內容策展模式（轉載 + 加值）
- **討論**: 我哋可以點用呢個模式？
- **Output**: 建立私人 YouTube 分析 skill

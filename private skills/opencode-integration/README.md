# OpenClaw Oh-My-Opencode 整合系統

> 將 Oh-My-Opencode 核心機制整合到 OpenClaw + ClawTeam 生態

## 系統概述

本專案實現四個核心機制：

1. **IntentGate** - 意圖分類與智能路由
2. **Hash-Anchored Edit** - 哈希驗證編輯系統
3. **Todo Enforcer** - 任務強制執行機制
4. **Plan 文件驅動** - 專案計劃管理

## 快速開始

```bash
# 安裝依賴
pip install -r requirements.txt

# 初始化系統
python scripts/init.py

# 運行測試
python -m pytest tests/
```

## 模組說明

### modules/intentgate/
意圖分類引擎，支持語義路由和 ClawTeam 代理映射

### modules/hash_anchored_edit/
哈希錨定編輯系統，提供可靠的文件編輯機制

### modules/todo_enforcer/
任務強制執行系統，與 AMS 深度整合

### modules/plan_driven/
Plan 文件驅動專案管理系統

## 配置

編輯 `config/integration.yaml` 自定義系統行為

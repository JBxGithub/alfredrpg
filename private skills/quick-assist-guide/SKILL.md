# Quick Assist 遠端協助指南

> **用途**: 使用 Windows 快速助手 (Quick Assist) 進行遠端協助
> **創建日期**: 2026-04-15
> **作者**: ClawTeam

---

## 概述

Windows 快速助手 (Quick Assist) 是一個內建工具，允許用戶從另一部電腦遠端協助或控制當前電腦。

---

## 使用流程

### 被協助方（當前電腦）操作步驟

#### 步驟 1: 打開快速助手

```powershell
# 方法 1: 快捷鍵
Win + Ctrl + Q

# 方法 2: 開始功能表搜尋
開始功能表 → 搜尋 "快速助手" 或 "Quick Assist"
```

#### 步驟 2: 獲取協助

1. 在快速助手視窗選擇 **"Get help"**（取得協助）
2. 等待系統生成 **6 位安全性驗證碼**
3. 將此代碼告知協助方

#### 步驟 3: 批准連接

當協助方輸入代碼後，畫面會顯示：

```
┌─────────────────────────────────────┐
│     Allow screen sharing?           │
│                                     │
│  [ ] I understand the security      │
│      implications of sharing        │
│      my screen                      │
│                                     │
│  [Allow]        [Decline]           │
└─────────────────────────────────────┘
```

**操作**:
1. ☑️ 勾選「I understand the security implications of sharing my screen」
2. 點擊 **「Allow」** 按鈕

#### 步驟 4: 批准控制權限

當協助方請求控制時，畫面頂部會顯示：

```
┌────────────────────────────────────────────────────────────┐
│  Burt S. is requesting control    [Allow] [Deny] [Leave]   │
└────────────────────────────────────────────────────────────┘
```

**操作**: 點擊 **「Allow」** 授權對方控制

---

## 協助方（遠端電腦）操作步驟

1. 打開快速助手 (Win + Ctrl + Q)
2. 選擇 **"Help someone"**（協助他人）
3. 登入 Microsoft 帳戶
4. 輸入被協助方提供的 **6 位代碼**
5. 點擊 **Submit**
6. 等待被協助方批准連接
7. 點擊 **Request control** 請求控制權限
8. 等待被協助方批准控制

---

## 常見問題

### Q: 代碼過期怎麼辦？
**A**: 關閉快速助手重新打開，會生成新代碼。

### Q: 公司網絡無法連接？
**A**: 公司防火牆可能封鎖了快速助手，建議使用其他遠端工具如 TeamViewer 或 AnyDesk。

### Q: 如何結束連接？
**A**: 點擊畫面頂部的 **「Leave」** 按鈕即可斷開連接。

---

## 安全注意事項

⚠️ **只與信任的人分享屏幕**
- 快速助手會給予對方查看和控制你電腦的能力
- 切勿將代碼分享給陌生人
- 連接期間不要輸入敏感密碼

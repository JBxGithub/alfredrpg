## 2️⃣ 處理層（Data Processing）

### 2.1 Absolute Score（趨勢判斷）- 100%

| 組件 | 權重 | 計算邏輯 | 門檻/輸出 |
|------|------|----------|-----------|
| **Market Phase** | 20% | 7階段判斷（價格 vs 200MA/50MA/20EMA） | -10 至 +10 |
| **Components Breadth** | 15% | Advance/Decline + New High/Low + Breadth Ratio | -10 至 +20 |
| **Components Risk** | 10% | 板塊輪動 + 集中度 | -10 至 +10 |
| **Futures OI** | 10% | OI 變化 + 價格方向 | -10 至 +15 |
| **ETF Flow** | 10% | QQQ/TQQQ/SQQQ 資金流向 | -10 至 +10 |
| **ADX** | 20% | 趨勢強度 + 方向（+DI/-DI） | 過濾器 |
| **MACD** | 10% | MACD vs Signal + 柱狀圖 | 0 至 +10 |
| **動量** | 5% | 20日價格變化 | 0 至 +5 |

**Absolute Score 範圍**：0 - 100  
**門檻**：
- > 60：潛在做多區間
- < 40：潛在做空區間
- 40-60：觀望

#### Market Phase 詳細計算

| 階段 | 條件 | 分數 |
|------|------|------|
| 強牛市 | 價格 > 200MA > 50MA > 20EMA | +10 |
| 牛市 | 價格 > 200MA > 50MA | +7 |
| 震盪上行 | 價格 > 200MA，但 < 50MA | +3 |
| 震盪 | 價格在 200MA 附近 | 0 |
| 震盪下行 | 價格 < 200MA，但 > 50MA | -3 |
| 熊市 | 價格 < 200MA < 50MA | -7 |
| 強熊市 | 價格 < 200MA < 50MA < 20EMA | -10 |

#### ADX 邏輯（方案 A）

```
IF ADX < 20: → 觀望（趨勢太弱）
IF 20 ≤ ADX ≤ 25: → 過渡區間，觀望或維持現狀
IF ADX > 25 AND +DI > -DI: → 可做多，持倉多單續持
IF ADX > 25 AND +DI < -DI: → 可做空，持倉空單續持
```

#### Components Breadth 計算

```python
if breadth_ratio > 0.7:    # 強勢
    score += 20
elif breadth_ratio > 0.5:  # 中性偏多
    score += 10
elif breadth_ratio > 0.3:  # 中性偏空
    score += 0
else:                      # 弱勢
    score -= 10
```

## 3️⃣ 決策層（Decision Engine）

### 步驟式決策流程

```
┌─────────────────────────────────────────────────────────┐
│ 步驟 1: VIX 檢查（風險控制）                              │
├─────────────────────────────────────────────────────────┤
│  IF 持多單 AND VIX > 25:                                │
│     → 立即止損平倉                                      │
│     → 重新循環（回到步驟 1）                            │
│                                                         │
│  IF 持空單 AND VIX > 30:                                │
│     → 止盈縮短至 +3%                                    │
│     → 回升 1% 止損平倉                                  │
│     → 重新循環（回到步驟 1）                            │
└─────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────┐
│ 步驟 2: Absolute Score（趨勢判斷）                       │
├─────────────────────────────────────────────────────────┤
│  計算 Absolute Score（8 組件加權）                       │
│                                                         │
│  IF Score > 60:  → 潛在做多區間                         │
│  IF Score < 40:  → 潛在做空區間                         │
│  IF 40 ≤ Score ≤ 60: → 觀望，結束                       │
└─────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────┐
│ 步驟 3: ADX 確認（趨勢強度 + 方向）                       │
├─────────────────────────────────────────────────────────┤
│  IF ADX < 20: → 觀望，結束                              │
│                                                         │
│  IF 20 ≤ ADX ≤ 25: → 過渡區間，觀望或維持現狀            │
│                                                         │
│  IF ADX > 25 AND +DI > -DI: → 可做多                    │
│  IF ADX > 25 AND +DI < -DI: → 可做空                    │
└─────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────┐
│ 步驟 4: Reference Score（交易時機確認）                   │
├─────────────────────────────────────────────────────────┤
│  計算 Reference Score（7 組件加權）                      │
│                                                         │
│  IF Score > 55: → ✅ 確認做多 / 多單續持                 │
│  IF Score < 40: → ✅ 確認做空 / 空單續持                 │
│  IF 40 ≤ Score ≤ 55: → ⏸️ 觀望，結束                    │
└─────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────┐
│ 步驟 5: 執行交易                                         │
├─────────────────────────────────────────────────────────┤
│  根據信號執行：                                          │
│  - BUY_LONG（做多 TQQQ）                                 │
│  - BUY_SHORT（做空 SQQQ）                                │
│  - HOLD（維持現狀）                                      │
│  - SELL（平倉）                                          │
│                                                         │
│  觸發條件後 → 重新循環（回到步驟 1）                     │
└─────────────────────────────────────────────────────────┘
```

## 4️⃣ 執行層（Execution）

### 交易規則（繼承 V9 不對稱邏輯）

| 動作 | 條件 | 風控參數 |
|------|------|----------|
| **開多單 (TQQQ)** | Absolute >60 + ADX>25+DI>-DI + Reference >55 | 倉位上限 95% |
| **開空單 (SQQQ)** | Absolute <40 + ADX>25+DI<-DI + Reference <40 | 倉位上限 95% |
| **續持多單** | Reference >55 | 止損 -3%，回落 -3%，止盈 +15%，重估 7天 |
| **續持空單** | Reference <40 | 止損 +2%，反彈 +2%，止盈 +10%，重估 3天 |
| **平倉** | 止損/止盈/VIX觸發/Reference觀望 | 立即執行 |

### VIX 特殊處理

| 持倉 | VIX 條件 | 行動 |
|------|----------|------|
| 做多 (TQQQ) | VIX > 25 | 立即全部止損平倉，重新循環 |
| 做空 (SQQQ) | VIX > 30 | 止盈縮短至 +3%，回升 1% 止損平倉，重新循環 |

### 再循環機制
```
平倉後 → 冷卻期 1 天 → 重新進入決策流程（步驟 1）
```

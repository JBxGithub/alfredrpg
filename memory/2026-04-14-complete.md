## 🔌 API 端點

### DeFi API
- `GET /api/v1/chains` - 鏈列表
- `GET /api/v1/protocols` - 協議列表
- `GET /api/v1/tvl/history` - TVL歷史
- `GET /api/v1/yields` - 收益率
- `GET /api/v1/yields/top` - 最高收益

### World Monitor API
- `GET /api/v1/world/conflicts` - 衝突事件
- `GET /api/v1/world/disasters` - 自然災害
- `GET /api/v1/world/risk-score` - 綜合風險評分

### Fusion API
- `GET /api/v1/fusion/signals` - 綜合信號
- `GET /api/v1/fusion/risk-score` - 融合風險評分
- `GET /api/v1/fusion/allocation` - 資金調配建議

## 🚀 運作模式

### 自動化流程
```
每 5 分鐘:
  ├─ 收集 FutuTradingBot 數據 (市場情緒)
  ├─ 收集 DeFi Intel 數據 (資金流向)
  ├─ 收集 World Monitor 數據 (宏觀風險)
  ├─ 信號融合計算 (加權算法)
  ├─ 生成綜合風險評分
  ├─ 判斷決策 (機會/風險/持有)
  └─ 執行動作 + 發送通知
```

### 決策邏輯
- **風險評分 > 70**: 減少敞口，轉向避險
- **風險評分 < 30**: 增加敞口，積極進取
- **套利機會出現**: 自動調配資金執行

# QQQ/TQQQ 套利策略可行性驗證

## 研究日期
2026-04-04

## 核心問題
QQQ/TQQQ 統計套利是否真的可行？

## 關鍵發現

### 1. TQQQ 的本質
- TQQQ 是 **3倍日內槓桿** ETF，不是簡單的 QQQ × 3
- 長期持有會有 **複利衰減**（volatility decay）
- 價格關係是非線性的，受波動率影響

### 2. 套利的根本問題
**問題：沒有自然的收斂機制**
- 股票對套利（Pair Trading）需要兩者價格有回歸關係
- QQQ/TQQQ 的價格偏離 **不會自動修復**
- TQQQ 可以長期偏離「理論價值」而不回歸

### 3. 實際風險
- TQQQ 的 3倍槓桿意味著 **3倍波動**
- 價格偏離可能持續數天甚至數週
- 保證金要求會隨波動急劇變化
- 極端行情下可能面臨 **追加保證金或強制平倉**

### 4. 歷史表現
根據公開資料：
- TQQQ 在 2020-2022 年間有巨大回撤（-90%+）
- 與 QQQ 的價格比率並不穩定
- 沒有明顯的均值回歸特性

## 結論

**⚠️ 此策略風險極高，不建議實施**

原因：
1. 沒有價格收斂的保證
2. 槓桿產品的複雜性
3. 保證金風險難以控制
4. 歷史數據不支持穩定盈利

## 建議

改為研究：
1. **行業輪動策略**（科技 vs 其他行業 ETF）
2. **跨市場套利**（如 QQQ 盤前盤後價差）
3. **波動率策略**（VIX 相關產品）
4. **傳統的配對交易**（如兩隻高度相關的股票）

## 教訓
不要基於「理論關係」做假設，必須有：
- 明確的價格收斂機制
- 歷史回測支持
- 可控的風險參數
